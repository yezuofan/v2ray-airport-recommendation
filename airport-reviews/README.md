## 🔍 如何选择机场

选购机场前，先问自己三个问题：

| 问题 | 推荐选择 |
|------|---------|
| **你主要用来做什么？** | 看Netflix/Disney+ → 选流媒体解锁强的；日常浏览 → 入门款够用 |
| **你在哪个地区使用？** | 大陆 → 优先IEPL专线；海外 → 看回程线路 |
| **你对稳定性要求多高？** | 生产力工具 → 别省这点钱；备用 → 买最便宜的 |

### 选机场最重要的指标（按优先级排序）

1. **稳定性** — 能用是最基本的，三天两头连不上再便宜也是浪费
2. **晚高峰表现** — 白天快没用，晚上8-11点速度才是真实水平
3. **流媒体解锁** — 需要看Netflix/ChatGPT，选之前先试
4. **售后响应** — 有客服和没客服的机场是两个世界

### ⚠️ 风险提示

- 机场有跑路风险，**不要一次性充值太多**，最多充够3个月用
- 尽量选**运营1年以上**的，踩坑概率低很多
- **不要把所有鸡蛋放一个篮子**，建议1个主力 + 1-2个备用

## ❓ 常见问题

**Q: 哪个机场最稳定？**

A: 目前自由猫 Freecat 是我测试下来最稳的。叶老大自己用了超过半年，中间没有出现连续3天以上不可用的情况。WgetCloud 和 NXO Earth 次之。

**Q: 为什么有些机场白天快晚上卡？**

A: 这是"中转线路"的通病。白天用的人少带宽够，晚上大家都在用就挤成停车场。IEPL专线是独享带宽，几乎不受影响，但价格也更贵。

**Q: 流媒体解锁是什么意思？为什么重要？**

A: Netflix/ChatGPT/Disney+ 等平台有地区版权限制，不同IP所在地能看的内容不同。好的机场会专门优化节点IP让你能解锁这些平台，普通机场的IP往往已经被平台封禁。

**Q: 注册链接和直链有什么区别？**

A: 注册链接带AFF返佣码，通过这个链接注册我能获得佣金（从你的首年费用里扣，不影响你的价格）。这是支持我持续测试和推荐的动力，感谢支持！

**Q: 节点速度慢怎么办？**

A: 晚高峰（8-11点）建议换香港/日本节点，美国节点延迟较高。如果所有节点都慢，说明这个机场容量不够，换备用机场。

**Q: 机场会跑路吗？**

A: 会。尤其是新机场和价格异常便宜的。防范方法：不要一次充太多、选择有运营历史的、关注机场的社群动态。
