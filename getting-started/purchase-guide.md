# 🛒 机场购买完整教程

还不确定选哪个？先看 **[场景选购指南](./how-to-choose.md)**，按你的需求匹配。

**本教程以「自由猫 Freecat」为例**，其他机场流程基本一样。

## 第二步：注册账号

1. 点击注册链接，进入官网
2. 用**手机号或邮箱**注册（建议用真实信息，客服找你时有用）
3. 注册完成后，登录控制台

**注意事项：**
- 不要使用身份证等敏感信息注册
- 建议设置强密码，或绑定 Google 二次验证
- 保存好注册手机号/邮箱，换设备登录需要

## 第三步：购买套餐

### 3.1 选择套餐

常见套餐类型：

| 套餐 | 流量 | 适合人群 |
|------|------|---------|
| 月付 | 100-200G | 先测试，好用再续 |
| 季付 | 300-500G | 正常日常使用 |
| 年付 | 不限/1000G+ | 长期使用，通常最划算 |

**建议：先月付测试 7-15 天，确认稳定再买长期套餐。**

### 3.2 付款方式

大多数机场支持：
- **支付宝**（最常用）
- **微信支付**
- **USDT（TRC20）**（匿名性最好）

### 3.3 付款流程

1. 选择套餐 → 点击购买/续费
2. 选择支付方式 → 扫码支付
3. 支付成功后，套餐即时到账

**防坑提醒：**
- ❌ 不要买远低于市场价的套餐（机场要运营成本，太便宜大概率要跑路）
- ❌ 不要一次买超过 3 个月的套餐（新机场尤其不要）
- ✅ 优先选支持 USDT 的机场（跑路风险相对低，支付也有记录）

## 第四步：获取订阅链接

套餐购买成功后，需要获取**订阅链接**导入到客户端。

### 通用步骤：

1. 登录机场控制台
2. 找到「**订阅**」或「**订阅管理**」菜单
3. 复制你的**专属订阅链接**（带 AFF 码的链接）

**订阅链接长这样：**
```
https://freecat.cloud/api/v1/client/subscribe?token=xxxxx
```

### 注意事项：

- 订阅链接**不要分享给他人**，相当于你的账号密码
- 如果链接失效，登录控制台重新生成
- 订阅链接一般有效期到账号过期那天

## 第五步：导入客户端

### 推荐客户端：

| 客户端 | 平台 | 推荐指数 | 说明 |
|--------|------|---------|------|
| **Clash Verge** | Windows/Mac | ⭐⭐⭐⭐⭐ | 界面简洁，功能全面 |
| **Clash for Windows** | Windows | ⭐⭐⭐⭐ | 功能强，但更新少 |
| **Stash** | iOS | ⭐⭐⭐⭐⭐ | iOS 上最好的 Clash 客户端 |
| **Shadowrocket** | iOS | ⭐⭐⭐⭐ | 经典应用，稳定好用 |
| **v2rayN** | Windows | ⭐⭐⭐ | V2ray 系，适合高级用户 |

### 使用订阅链接导入（以 Clash Verge 为例）：

1. 下载并安装 [Clash Verge](https://github.com/clash-verge-rev/clash-verge-rev/releases)
2. 打开软件 → 找到「**订阅**」→「**导入**」
3. 粘贴订阅链接 → 点击确定
4. 选择一个节点 → 开启系统代理

### 没有订阅功能？手动导入

如果客户端不支持订阅链接：

1. 在机场控制台手动获取**节点信息**（节点名称、地址、端口、UUID、加密方式）
2. 在客户端手动添加节点，填入对应信息
3. 建议优先使用 **VLESS + Reality** 协议（最稳、防封能力强）

## 常见问题

### Q: 支付成功了但套餐没到账？

**处理方法：**
1. 等 5-10 分钟（有些机场是异步到账）
2. 登录控制台 → 查看「我的套餐」确认状态
3. 找客服（保留支付截图）

### Q: 订阅链接打不开？

可能原因：
- 机场服务器波动，等几分钟再试
- 订阅链接过期，重新生成
- 被防火墙拦截，换个网络环境试试

### Q: 节点能连上但网速很慢？

参考 **[常见问题排查](../troubleshooting/common-issues.md)**：
1. 晚高峰换香港/日本节点
2. 尝试不同的协议（VLESS > VMess）
3. 检查本地网络是否正常

### Q: 想换其他机场，原来的套餐怎么办？

- 机场套餐一般**不可退款**
- 建议用完当前套餐再换
- 或者当作备用机场保留
