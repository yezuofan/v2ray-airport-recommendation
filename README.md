## 📚 教程

### 什么是 V2ray / Clash / SSR？

- **V2ray**：新一代翻墙协议，抗干扰能力强，隐蔽性好
- **Clash**：代理客户端，支持多种协议，统一管理规则
- **SSR**：ShadowsocksR，原版 SS 的分支，有混淆插件

### 如何选择？

| 场景 | 推荐协议 |
|------|----------|
| 日常浏览 / 社交媒体 | V2ray / SSR |
| ChatGPT / 流媒体 | V2ray + TLS |
| 游戏 / 低延迟需求 | Trojan / V2ray |
